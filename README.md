## Описание проекта

Рандомный питомец вылупляется из яйца нажатием по экрану. Есть 4 статы питомца, которые теряют -10hp каждые 5 секунд.

- **Голод**: покормить любимца можно едой из магазина, либо она может выпадать случайным образом в игре-тапалке
- **Энергия**: для восполнения энергии на максимум игрок должен уложить питомца спать на 1 минуту или дать ему зелье энергии
- **Счастье**: можно поиграть с тамагочи, нажимая на всплывающую еду в окошке
- **Чистота**: игрок может помыть питомца, поиграв в "тапалку"
Можно отвести питомца в туалет и нарядить его в аксессуары, купленные в магазине
## Реализация:
1. Система состояний питомца
- Питомец меняет внешность в зависимости от параметров (грустный, сонный, грязный)
- Комбинированные состояния: например, "грустный + сонный" имеет отдельную картинку
- Это реализовано через приоритетную цепочку условий в методе update_appearance()

2. Мини-игры как способ восстановления параметров
- Тапалка (игра): ловля предметов (рыба/кость/бамбук) повышает счастье и даёт монеты
- Мойка: ловля мыльных пузырей повышает чистоту
- Сон: восстанавливает энергию до 100 после 60 секунд
- Туалет: отдельная кнопка для сброса потребности

3. Экономическая система
- Монеты зарабатываются в тапалке (1 пойманный предмет = 1 монета)
- В магазине можно купить аксессуары (шляпа 50 монет, очки 30 монет)
- Зелье бодрости (105 монет) пробуждает питомца и даёт +30 энергии

4. Потребности питомца 
- Потребность во сне возникает, если энергия ниже 70 (проверка каждые 45 секунд)
- Потребность в туалете возникает каждые 60 секунд и ускоряет падение чистоты
- Пока питомец хочет спать — нельзя играть в тапалку

5. Аксессуары с разными позициями
- Шляпа и очки надеваются по-разному на каждое животное (индивидуальные координаты)
- Покупка сохраняется навсегда (флаги hat_owned, `glasses_owned`)

6. Адаптивный интерфейс
- В тапалке меняется предмет в зависимости от животного (рыба/кость/бамбук)
- Размер области клика подстраивается под размер предмета
- В режиме сна отображается спящая версия текущего питомца

## Техническая реализация
- Архитектура: разделение на модули (main, pets, ui, shop, tap_game, cleaning_game)
- ui.py - все элементы интерфейса
- Таймеры: все изменения статов происходят через pygame.time.get_ticks() без блокировки основного цикла
- Сохранение состояния: через глобальные переменные внутри классов

## Использование нейросетей

Нейросети использовались как вспомогательный инструмент при разработке проекта:

- Для поиска несостыковок в разных кодах участников
- Для поиска ошибок, связанных с разным названием одних переменных
- Для генерации изображений питомцев, фонов, аксессуаров, кнопок
- Для объединения кодов участников
- Для улучшения структуры кода(компактности) и оформления README

## Как запустить игру

### Требования
- Python 3.8+
- Pygame
### Установка
1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/tamagotchi.git
cd tamagotchi
```
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Запустите игру:
```bash
python main.py
```

### Управление

- **Кнопка "Кормить"** — снижает голод
- **Кнопка "Играть"** — запускает TapGame, увеличивает счастье и монеты
- **Кнопка "Спать"** — питомец засыпает на 60 сек, восстанавливает энергию
- **Кнопка "Мыться"** — запускает CleaningGame, увеличивает чистоту
- **Кнопка "Туалет"** — убирает потребность в туалете
- **Кнопка "Магазин"** — открывает магазин для покупки аксессуаров
